ng-idle-forms

directives and services for creating forms

Feature Usage Description
AutofocusDirective <input [autofocus] /> Sets focus to the control
FormValidators.range FormValidators.range(1,99) Validate the input is a number between the specified range
FormValidators.restrictedWords FormValidators.restrictedWords(['foo','bar']) Validates the specified words aren't in the input value
ValidationForm
An abstract class that provides easy validation for reactive forms. Set the validationMessages in the constructor and build the form in ngOnInit.
