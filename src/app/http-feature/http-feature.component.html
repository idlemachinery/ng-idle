ng-idle-http

the http service

Feature Description Usage
AddHeaderInterceptor
Intercepts http calls and adds Content-Type: application/json to the headers.
AddXmlHeaderInterceptor
Intercepts http calls and adds Content-Type: application/xml to the headers.
GenericDataService
An abstract generic class that a data service can inherit from. Has built in CRUD functionality using http get, post, put and delete.
